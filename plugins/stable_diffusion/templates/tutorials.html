图片生成命令教程

图片生成主要使用的就是 #sd draw 命令，在它后面可以跟随很多参数来控制生成结果：

参数 含义 默认值 描述
-p <tokens> 正向提示词 "" 一个使用英语双引号括住的字符串
希望出现在图中的关键词，如："1girl,solo"
-n <tokens> 负向提示词 "" 一个使用英语双引号括住的字符串
不希望出现在图中的关键词，如："nsfw,low res"
-n <steps> 迭代步数 25 一个 20 到 40 的整数
步数越少，图片越看不出细节
-s <seed> 种子 随机整数 一个正整数
当其他参数不变时，固定种子可以保证生成完全一样的图
-r <ratio> 图片宽高比 "1:1" 一个使用英语双引号括住的比例数
图片以 512x512 为基础面积，然后通过宽高比进行拉伸得到最终宽高
-h 禁用高分辨率 默认情况下插件会开启图片高分辨率，你可以在图片调教的前期使用这个参数关闭高分辨率来加快生成速度
-S <scale> 高分辨率放大倍数 2 一个 1 到 3 之间的实数
倍数越高生成越慢，得到的图片也越大
-I <steps> 高分辨率迭代步数 20 一个 0 到 20 的整数
越低图片细节不足，越高图片容易出现畸形
-d <denoising> 高分辨率重绘幅度 0.2 一个 0 到 1 之间的实数
越低放大后与原图更接近，越高和原图差异变大

 

Lora 使用

Lora 可以帮助你画出你想要的元素，如果你的提示词总是得不到想要的，那么不妨来试试 Lora。

在使用 Lora 前，你需要输入命令 #sd lora-menu 查询插件支持的 Lora 列表。

在物色到你想要的 Lora 之后，你需要在图片生成的 正向提示词 中按照如下格式填写一个标签：

<lora:Lora 的名字/别名:Lora 的权重>

如果一个 Lora 拥有 别名，那么我强烈建议你填写别名，因为有些 Lora 的全名非常长，还有可能意义不明，而由插件管理员设置的别名往往更短，且意义更加明确。

假如有一个支持的 Lora 它的名字叫“lappland”，我需要让它的权重为 0.8，则我需要填写：<lora:lappland:0.8>

在你的绘画请求开始被处理时，插件会自动分析你的 Lora 是否受到支持，并发送一段文字告知你，你可以由此来检查你的 Lora 是否正确被使用。
